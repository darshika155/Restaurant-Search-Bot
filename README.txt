RASA Restaurant Search Chatbot


Installations
-Rasa core and NLU(Version: 1.0.0)
-Rasa dependencies for spaCy(Version: 2.1.9)
-Smtplib
-Requests 
-Json
-Re (Regular expression operations)

Versions
There are two models in the models folder.
1.	20200614-032126.tar.gz
2.	20200613-182348.tar.gz
The latest model is 20200614-032126.tar.gz
Running    $rasa shell    will automatically use the latest model.

Instructions for Running
1.	Open terminal in the “Rasa_basic_folder – final” location.
2.	Run $rasa run actions
3.	Open another terminal in the same window and run $rasa shell
4.	Start your conversation with the Chatbot!!
